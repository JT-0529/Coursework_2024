Lab: FishNet Sample Client (Modified)
This lab modifies the client program from Lab 10 to access and manipulate devices in the FishNet simulated network environment. Key tasks include:

Logging into the FishNet server.
Obtaining a token for access.
Listing the full device inventory and fetching the configuration of switch1.
Modifying the IP address of switch1.
Verifying that the IP address was successfully modified by comparing the new IP address.
Requirements:
Python 3.x virtual environment with Flask, Flask_api, and requests packages installed.
Internet access to GitHub for repository access.
FishNet server running and accessible.
