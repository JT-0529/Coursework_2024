Lab 04: Configure File and Share Access
In this lab, you will configure file and share access on a Windows Server 2012R2 machine. Key tasks include:

Creating SMB file shares
Setting custom share permissions
Enabling NTFS quotas
Configuring Volume Shadow Copy Services (VSS) for backup
Setting up Work Folders for file synchronization





Lab 18: Application Restriction Policies
This lab explores the implementation of software restriction policies and AppLocker to control application access on a domain. It includes:

Installing software via Group Policy
Configuring software restrictions and testing them
Using AppLocker for enhanced application control and security
