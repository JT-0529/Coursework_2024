Module 4 Lab 4: SNMP Operations Using a MIB Browser (Network Setup)
In this lab, students will learn to perform SNMP operations across multiple routers in a network. The tasks include:

Configuring SNMP on Cisco routers R1, R2, and R3.
Verifying connectivity from SNMP manager PC to network devices.
Using MIB Browser to query SNMP data from devices and change system contact information.
Performing SNMP GET and SET operations on network devices in a live network environment.
